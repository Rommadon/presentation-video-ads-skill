# Examples

These examples show how to use the skill in different starting conditions.

## Files

- `empty-project.md` — start from nothing
- `about-us.md` — transform an about page
- `pricing.md` — transform a pricing or package page
- `react-implementation.md` — the default current-app implementation prompt

The prompts now favor more, shorter scenes, one focus per scene, visible UI states, modern product mockups, and a minimal transport bar.

## How to use them

Use the file that matches your starting point, then swap in your own project content.
