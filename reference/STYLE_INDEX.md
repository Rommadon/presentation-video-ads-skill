# Style Index

This is the lightweight style chooser for `presentation-feature-video-ads`.

Read this file first when the user needs a visual direction. Then read `templates/index.json` for compact metadata. Read only the shortlisted template `preview.md` files. Read a full `design.md` only after the user picks a template.

## Shared authority

These files apply to every template in the pack:

- `reference/STYLE_GUIDE.md` — house-style contract shared across the pack
- `reference/scene-grammar.json` — shared scene sequencing and pacing rules

## Starter templates

| Template | Use when | Avoid when |
| --- | --- | --- |
| `presentation-feature-core` | You want the strongest default: cinematic, premium, UI-first, product-marketing ad energy | You need a quieter editorial tone or a radically different print-inspired surface |
| `soft-editorial` | You want a softer paper/editorial tone while keeping the same ad pacing | You need darker contrast or sharper enterprise drama |
| `emerald-editorial` | You want magazine-cover business energy with confident premium framing | You need a playful or understated atmosphere |
| `vellum` | You want a quiet, scholarly, dark editorial system | You need bright pop color or louder commercial energy |
| `capsule` | You want modular, rounded, friendly surfaces with premium clarity | You need institutional seriousness or hard-edge austerity |
| `cobalt-grid` | You want design-research precision with sharp typographic structure | You need warmth, softness, or playful motion |

## Loading rule

1. Read `templates/index.json`.
2. Shortlist up to 3 candidate templates from metadata only.
3. Read only those templates' `preview.md` files.
4. After the user chooses, read exactly one `design.md`.
5. Use template-local example briefs only for the selected default/base template when they help the current job.
